What is HTML? How is HTML used? Logo

HTML is a launguage used by web devleopers to make websites. This is because browsers can interpret HTML and render a page

HTML uses tags, such as <img> and <table>. these tags are interpreted by the browsers to do things, like adding an image or creating a table. The tags can have attributes in them, like id="" (id of the tag) or href="" (url for linking between pages, or websites)
