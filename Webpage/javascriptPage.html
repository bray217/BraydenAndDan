What is JavaScript? How is JavaScript used? Logo

JavaScript is a scripting language standardised by ECMA and their EMCAScript program

JavaScript has many uses, due to how high level of a programing language it is, but is is mostly used in web development, both server and client side. JavaScript can be simply added to HTML in <script> tags. The contents of these tags are not displayed but executed by the browser as JavaScript code.
