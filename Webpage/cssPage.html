What is CSS? How is CSS used? Logo

CSS is a language used to style the HTML that the browser is displaying

CSS generally has it's own page, a "Stylesheet", that the code is writen in. This is linked at the top of an HTML document. There are HTML tags on the Stylesheet, or ID's of certain things, such as body or id="content", CSS code, such as font-size or text-align, tells the tags or elements how to look, or behave, on the webpage.
